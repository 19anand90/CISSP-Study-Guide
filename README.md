# CISSP
study material used for the 2018 CISSP exam

# About the exam:
Need 5 years expierence for certification.  Get associate if less than that.<br />
3 hours <br />
100-150 Multiple Choice Questions <br />
70% to pass <br />
All domains 10-15% of score <br />
